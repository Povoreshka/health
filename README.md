# Health Fitness App

Веб-приложение для мониторинга здоровья и автоматизированного формирования программ физических упражнений на основе индивидуальных показателей пользователя.

## Стек

Frontend:
- React
- React Router
- Axios

Backend:
- Node.js
- Express
- PostgreSQL
- bcryptjs

## Основные функции

- регистрация и вход пользователя;
- многоэтапный онбординг;
- расчёт ИМТ;
- генерация недельного расписания тренировок;
- выполнение тренировки с таймером;
- дневник здоровья;
- статистика прогресса;
- достижения;
- создание пользовательских тренировок.

## Запуск backend

- cd backend
- npm install
- npm start

## Запуск frontend

- cd frontend
- npm install
- npm run start
